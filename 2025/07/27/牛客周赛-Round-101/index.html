# A 题解的 token 计算

调用自带的 logloglog 函数，默认以 eee 为基底

# B 76 修地铁

对每个要求考虑消耗量即可

# C 76 选数

由于是按位异或操作，不难发现能得到的最大整数一定是从给定上界的最高位到最低位都赋值为 1

# D 76 构造

既然需要每个区间的元素 gcdgcdgcd 按位异或起来等于整数 mmm ，不妨先将 mmm 按二进制拆分

我们把拆出来的数分别看作一个区间，这些区间按位异或起来就是 mmm

如果 mmm 是奇数，剩下的区间合并进拆出来的 1，否则剩下的区间必然会改变原有的状态，即无解

# qcjj 寄快递

不难发现路径顺序是固定的，我们只需要关系每两个点之间的耗时

设缩放屏幕的时间为 kkk ，那么距离变成 $\fracdis} {2(k)$ ，总时间花费为 $2 \times k + \fracdis} {2({k+1)} $

显然 / 求导看出这是一个单峰函数

用三分法解决即可

# 幂中幂 plus

本次计算的结果会变成下一次计算的指数，不难发现 kkk 的值很大但是 modmodmod 的值被限定到了 10610^6106

从 modmodmod 入手，不难发现计算结果最多有 modmodmod 种

所以必然存在循环节

那么我们只需要暴力模拟最多 10610^6106 次计算，找出循环节后根据循环节计算答案即可
